SISTEMA DE CONTROL DE VENTAS - EL SALVADOR
-------------------------------------------
DESCRIPCIÓN:
Este proyecto permite a comerciantes de mercados locales llevar un control organizado 
de sus ventas diarias. El sistema calcula ingresos totales y por producto, mostrando 
reportes en pantalla.


CARACTERÍSTICAS:
- Registro de ventas diarias (producto, precio, cantidad).
- Cálculo automático de ingresos totales.
- Reporte de ingresos por producto.
- Resultados ordenados por mayor ingreso.
- Incluye 3 versiones distintas del sistema:
  1. Versión 1: Listas y diccionarios.
  2. Versión 2: Clases y objetos.
  3. Versión 3: Diccionarios anidados.

ESTRUCTURA DEL PROYECTO:
El proyecto puede usarse en un solo archivo (ventas.py) o separado en módulos.

proyecto/
│── ventas.py     -> Código principal con las 3 versiones integradas
│── README.txt    -> Este archivo con instrucciones

CÓMO EJECUTAR:
1. Tener instalado Python 3.
2. Abrir la terminal en la carpeta del proyecto.
3. Ejecutar el programa con:
   python ventas.py
4. Elegir la versión que se desea usar:
   - 1: Listas y diccionarios
   - 2: Clases y objetos
   - 3: Diccionarios anidados
